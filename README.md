
# Corona PWA

This app shows the covid cases in India.

The data shown here is sourced from mohfw.gov.in.
We do not verify the sources and correctness of the data. The data sourced from govt website is highly optimised for performance.

This is completely for educational purposes and for non commercial use. 


# Technologies Used


## Backend

* Nodejs with Express 
* Cheerio for web scrapping
* In memory caching using node-cache


## Frontend

* React project setup using webpack + babel
* Material UI for design
* Service worker implemented

## Demo

http://tracky.online

## Setup

Backend setup

```bash
cd backend
npm install
npm run start
```

Frontend Setup 

```bash
cd frontend/corona-pwa
npm install 
npm run start
```
